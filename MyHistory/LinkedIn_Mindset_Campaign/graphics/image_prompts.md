# Image Prompts Index — THE ORDER (AI Mindset campaign)

Each post's full graphic specification lives in its post folder as `graphic_brief.txt` — including the primary text-capable prompt, a no-text Midjourney fallback, overlay copy, and alt text. This file holds only what's shared.

## Visual system (deliberately distinct from the Thursday campaign)

The Modernization campaign's graphics live in dark, forensic, enterprise territory (receipts, code, server rooms). The Mindset campaign is about PEOPLE changing behavior, so its system is warmer and more metaphorical:

- Same brand palette (Purple `#4D1D82`, Orange `#E8640A`, Dark `#1A0A2E`, Light `#F8F6FC`) — one brand, two moods.
- Mindset graphics alternate light and dark backgrounds; Modernization is almost always dark. In the feed, this rhythm signals "different series, same person" without reading the caption.
- Orange always marks the changed behavior / the target state (Level 3, the first-step pill, the rod). Purple marks the status quo.
- Metaphor over abstraction: a staircase, a wall, a stopwatch, a cape, a rod — every image is a scene you could describe on the phone. No generic "glowing AI networks," ever. If the image could illustrate any AI post on LinkedIn, it fails.
